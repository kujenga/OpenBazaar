read(self, template)

source code 

Non-destructively reads a tuple in the tuple space.

This operation is similar to "get" (or "in") in that the peer builds a template and waits for a matching tuple in the tuple space. Upon finding a matching tuple, however, it copies it, leaving the original tuple in the tuple space.

Note: This method is named "rd" in some other implementations.

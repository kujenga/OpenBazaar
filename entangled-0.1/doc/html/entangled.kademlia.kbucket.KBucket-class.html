getContacts(self, count=-1, excludeContact=None)

source code 

Returns a list containing up to the first count number of contacts

Parameters:
  • count (int) - The amount of contacts to return (if 0 or less, return all contacts)
  • excludeContact (kademlia.contact.Contact or str) - A contact to exclude; if this contact is in the list of returned values, it will be discarded before returning. If a str is passed as this argument, it must be the contact's ID.
Returns: list
Return up to the first count number of contacts in a list If no contacts are present an empty is returned
Raises:
  • IndexError - If the number of requested contacts is too large

keyInRange(self, key)

source code 

Tests whether the specified key (i.e. node ID) is in the range of the 160-bit ID space covered by this k-bucket (in otherwords, it returns whether or not the specified key should be placed in this k-bucket)

Parameters:
  • key (str or int) - The key to test
Returns: bool
True if the key is in this k-bucket's range, or False if not.

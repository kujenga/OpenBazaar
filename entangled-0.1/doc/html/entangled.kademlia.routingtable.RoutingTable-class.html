Interface for RPC message translators/formatters

Classes inheriting from this should provide a suitable routing table for a parent Node object (i.e. the local entity in the Kademlia network)

findCloseNodes(self, key, count, _rpcNodeID=None)

source code 

Finds a number of known nodes closest to the node/value with the specified key.

Parameters:
  • key (str) - the 160-bit key (i.e. the node or value ID) to search for
  • count (int) - the amount of contacts to return
  • _rpcNodeID (str) - Used during RPC, this is be the sender's Node ID Whatever ID is passed in the paramater will get excluded from the list of returned contacts.
Returns: list
A list of node contacts (kademlia.contact.Contact instances) closest to the specified key. This method will return k (or count, if specified) contacts if at all possible; it will only return fewer if the node is returning all of the contacts that it knows of.

getRefreshList(self, startIndex=0, force=False)

source code 

Finds all k-buckets that need refreshing, starting at the k-bucket with the specified index, and returns IDs to be searched for in order to refresh those k-buckets

Parameters:
  • startIndex (index) - The index of the bucket to start refreshing at; this bucket and those further away from it will be refreshed. For example, when joining the network, this node will set this to the index of the bucket after the one containing it's closest neighbour.
  • force (bool) - If this is True, all buckets (in the specified range) will be refreshed, regardless of the time they were last accessed.
Returns: list
A list of node ID's that the parent node should search for in order to refresh the routing Table
